HuefyConfiguration

Configuration data class for the Huefy client.

apiKey: String - Your Huefy API key
timeout: Long = 30000 - Request timeout in milliseconds
retryAttempts: Int = 3 - Number of retry attempts
retryDelay: Long = 1000 - Initial retry delay in milliseconds
enableLogging: Boolean = false - Enable request/response logging
